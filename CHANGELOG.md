## Versioning Policy

POS Ledger NG follows the Semantic Versioning (SemVer) model.

- **MAJOR** version: Breaking changes.
- **MINOR** version: New features added without breaking compatibility.
- **PATCH** version: Bug fixes and performance improvements.

Example:

- 1.0.0 – Initial Release
- 1.1.0 – New Features
- 1.1.1 – Bug Fixes
- 2.0.0 – Major Platform Upgrade
